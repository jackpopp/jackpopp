Griddle

css/sass

Griddle is a lightweight, customisable, sass based grid system. The grid system comprimises of desktop, tablet and mobile grids aswell as visiblity classes.

Variables can be changed in the source file to create new builds, for example change the column amount from 8 to 12.

Dside

javascript/coffeescript

A javascript event dispatcher library built with coffeescript, designed to construct object and fire events based on the current browser url.

It allows you to write page specific code that will be executed only on that page and also profiles filters to run code on every page.

Syml PHP

php

A simple php mvc framework with RESTful routing, IOC container a simple ORM and more.

The goal of the project was to get a better understanding of frameworks such as laravel and to create something half way between a Slim PHP and Laravel.

Fflog

php/laravel

Fflog is a laravel based flat file blogging platform, it is deisgned to be very stripped back with a simple theming system

The project is currently built within an installation but will be refactored to become a package that can be added to any installation of laravel.

Tura

php/laravel

Tura is a laravel package that allows you to use named routes in your frontend javascript code.

Named routes can be set to be accessable via a tura javascript object by setting a key when registering a route.
